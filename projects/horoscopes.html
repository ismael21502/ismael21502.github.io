Overview

A simple horoscope web application built with Django. The site displays a list of zodiac signs, and each sign has its own detail page showing the horoscope of the day. The content is powered by a Django model, making the application easily extendable and maintainable.

Technologies used

HTML
CSS
Django

Key features

Dynamic content with Django models: Zodiac signs, images, date ranges, and daily predictions are stored and retrieved from the database.

Quick UI: List of zodiac signs with corresponding images for quick navigation.

Daily horoscope display: Each sign has its own dedicated page showing the horoscope of the day.

Scalable structure: Easily extendable to support weekly/monthly horoscopes, user personalization, or additional zodiac-related content.
