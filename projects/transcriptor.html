Audio Transcriptor App

Overview

It is a desktop application designed to simplify and automate the transcription of multimedia files. The tool allows users to select an audio or video file, process it, and send the data to AssemblyAI services to obtain a high-precision transcription. It includes key features such as an integrated audio player to select specific fragments of the file for transcription, and the ability to export the final result to a .docx fileâ€”optimizing the workflow from multimedia content to text documents.

Technologies used

Python
Flet
AssemblyAI
Pydocx
Multi-threading

Key features

Multimedia support: Accepts both audio and video files, converting them automatically into the required format.

Audio player: Enables users to select and transcribe only specific fragments of the file.

Export to Word: Generates a .docx document with the final transcription for easy editing and later use.

Transcription visualization and editing: Displays the transcribed content directly in a text field within the interface.

Real-time progress tracking: A progress bar shows the percentage of completion in real time.

Desktop notifications: Alerts the user with a notification once the transcription process is complete.
